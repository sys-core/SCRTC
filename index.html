Bienvenue à la page de notre outil SCRTC

Nous vous présentons deux scripts, un pour l'injection de code et l'autre pour l'injection Dll. Vous pouvez utiliser l'injection de code pour tester un morceau de shellcode conçus pour tuer un processus basé sur son PID.

Comment l'utiliser le script d'injection de code

Prenez le script pour un spin en démarrant un couple de processus cmd.exe, obtenez leurs PID respectifs, et transmettez les comme arguments de ligne de commande, comme ceci:

./SCCODE_INJECTEUR.py

Comment l'utiliser le script d'injection DLL

La DLL doit être chargée, dans le processus, et vous devriez voir une boîte de dialogue pop-up qui indique que la DLL est entrée dans le processus. Utilisez le script comme ceci:

./SCDll_Injecteur.py <Chemin de la DLL
